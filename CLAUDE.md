# CLAUDE.md

## Language policy

This repository is intended to become public alongside the dotmon library.

- All repository artifacts are written in **English**: commit messages, PR and issue titles/bodies, and code comments.
- Commit messages follow **Conventional Commits**.
- PR and issue bodies may add a Japanese translation in a collapsed `<details><summary>日本語</summary>` block.
- Existing Japanese code comments are translated to English opportunistically when the surrounding code is touched — no bulk rewrites.
- Conversation with the maintainer stays in Japanese; the artifacts produced from it are in English.
